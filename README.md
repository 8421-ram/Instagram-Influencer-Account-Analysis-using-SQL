# 📊 Instagram Influencer Account Analysis – SQL Project

## 📌 Overview
This project analyzes Instagram activity data of a tech influencer using SQL to uncover performance trends, measure account growth, and identify effective content strategies.

## ⚙️ Problem Statement
The objective of this project is to analyze Instagram data and:
- Explore unique post types and their engagement metrics  
- Identify highest & lowest impressions for each post type  
- Track profile visits and new followers month-over-month  
- Calculate total reach % by post type  
- Display post categories with monthly counts  
- Build a stored procedure to calculate total shares by week  

## 🛠️ Tools & Technologies
- SQL (MySQL)
- MySQL Workbench

## 📂 Dataset
- Instagram activity dataset (posts, impressions, reach, followers, etc.)

## 🔍 Key Insights
- 📈 Reels generated the highest impressions (339.7K)  
- 📉 Carousels recorded the lowest impressions (3,264)  
- 📊 May had the highest profile visits (23.79%)  
- 📉 February showed the lowest new followers (5.04%)  

## 💡 Key Features
- Advanced SQL queries for data analysis  
- Aggregations and grouping techniques  
- Month-over-month growth tracking  
- Stored procedure for weekly share analysis
